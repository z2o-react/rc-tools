## Usage

```bash
# compile(TS,JS)
$ rc-tools run compile --babel-runtime

# start dev server
$ rc-tools run storybook

# build examples
$ rc-tools run build

# build examples => gh-pages
$ rc-tools run gh-pages

# compile => npm publish => gh-pages => tag
$ rc-tools run pub
```
