## Usage

```bash
# 编译 TS 文件，启用babel-runtime
$ rc-tools run compile --babel-runtime

# 启动 storybook 开发模式
$ rc-tools run storybook

# 编译 storybook 生产模式
$ rc-tools run build

# 编译 storybook 并发布 gh-pages
$ rc-tools run gh-pages
```
